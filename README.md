# Страница отзывов 360shop

Готовая страница в стиле Telegram, которая загружает отзывы из Google Sheets и автоматически показывает новые записи.

## Что уже сделано

- Отзывы читаются напрямую из Google Sheets.
- Поддерживаемый формат строки:
  - `Имя` (или `name`)
  - `Отзыв` (или `текст`/`review`)
  - `Ссылка на телеграм` / `tg_username` (необязательно)
- Если в Telegram поле указано `@username`, собачка удаляется автоматически.
- Кнопка формируется как `https://t.me/username`.
- Автообновление страницы по таймеру (по умолчанию каждые 45 секунд).
- Встроен фильтр, который отсеивает случайные сообщения и оставляет только тексты, похожие на реальные отзывы.
- Можно вручную модерировать строки через дополнительную колонку: `is_review` / `approved` / `показывать`.
  - `1`, `true`, `yes`, `да` -> показать принудительно.
  - `0`, `false`, `no`, `нет` -> скрыть принудительно.

## Настройка

1. Откройте `config.js`.
2. Вставьте ID таблицы в `sheetId`.
3. Укажите название листа в `sheetName` (например, `Лист1`).
4. Убедитесь, что у таблицы открыт доступ:
   - `Открыть доступ` -> `Все, у кого есть ссылка` -> `Читатель`.

Пример:

```js
window.SHEET_CONFIG = {
  sheetId: "1AbCdEfGhIjKlMnOpQrStUvWxYz1234567890",
  sheetName: "Лист1",
  refreshIntervalMs: 45000,
  reviewFilter: {
    enabled: true,
    minScore: 2,
    minChars: 8,
    minWords: 2,
    maxDigitRatio: 0.35,
    maxDigitRun: 5,
    shortReviewMaxWords: 4,
    shortReviewMaxChars: 35,
  },
};
```

### Настройка строгости фильтра

- `minScore: 2` — сбалансированный режим (рекомендовано).
- `minScore: 3` — строгий режим.
- `enabled: false` — отключить фильтр и показывать все записи с текстом.
- Если всё еще пролезают числа/статусы, уменьшите `maxDigitRatio` до `0.25` и/или `maxDigitRun` до `4`.

## Запуск локально

Просто откройте `index.html` в браузере.

## Публикация на GitHub Pages

1. Загрузите файлы в репозиторий GitHub.
2. В репозитории откройте `Settings -> Pages`.
3. В `Source` выберите ветку (обычно `main`) и папку `/ (root)`.
4. Сохраните, дождитесь ссылки на сайт.

После этого страница будет всегда подтягивать свежие отзывы из таблицы.
